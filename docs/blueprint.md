# **App Name**: Hotel RIGUT

## Core Features:

- User Authentication: Securely authenticate users with username and PIN, validating credentials against a local data storage.
- Interactive Login Screen: Presents a visually appealing and intuitive login screen for user authentication.
- Registration Option: Allows new users to register with their credentials if they do not already have an account.
- Password Recovery: Enables users to recover their password via email.
- User Interface Personalization: Employing a tool that allows the app to choose appropriate visual elements that suit the particular user.

## Style Guidelines:

- Background color: Light pink (#F8E7EB), offering a soft and welcoming feel, conveying cleanliness and relaxation.
- Primary color: Rose-pink (#F472B6), a vibrant hue evoking positive feelings and aligned with contemporary design aesthetics; ensure it contrasts against the light background.
- Accent color: Lavender (#E699FF) to complement the pinks with a cool, modern contrast; use selectively for interactive elements.
- Font pairing: 'Poppins' for headlines and 'PT Sans' for body. 'Poppins' is a sans-serif suited to shorter, contemporary headlines. 'PT Sans' provides a readable, warm typeface for any longer text.
- Maintain a clean, uncluttered design with ample spacing between elements for improved readability.
- Incorporate smooth transitions and subtle animations on user interactions to enhance the experience.
